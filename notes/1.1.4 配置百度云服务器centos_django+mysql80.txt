CentOS7安装mysql80；
1.1.3.1启动MySQL服务 ( Starting the MySQL Server )

用以下命令启动MySQL服务器： ( Start the MySQL server with the following command: )


shell> sudo systemctl start mysqld.service


检查MySQL服务器的状态：
( You can check the status of the MySQL server with the following command: )


shell> sudo systemctl status mysqld.service
 


1.1.3.2安装的时候已经创建了超级用户帐户“root”@“localhost”。

用这个命令可以查看初始密码
sudo grep 'temporary password' /var/log/mysqld.log
 

1.1.3.3设置超级用户的密码并将其存储在错误日志文件中。要显示它，请使用以下命令：


shell> mysql -uroot -p


通过使用生成的临时密码登录并为超级用户帐户设置自定义密码，
尽快更改root密码：

mysql> ALTER USER 'root'@'localhost' IDENTIFIED BY 'MyNewPass4!';


这密码包含至少一个大写字母，一个小写字母，一个数字和一个特殊字符，并且密码总长度至少为8个字符。
新密码为：Lj@20091217
退出mysql: exit;
进入mysql: mysql -uroot -p，按enter后提示输入密码；
root为用户名，可变更；
